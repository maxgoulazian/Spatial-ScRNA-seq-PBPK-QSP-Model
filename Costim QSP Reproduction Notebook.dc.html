2. Per-cell receptor layer — panel-native + scVI overlay imputation + RNA→copies

This is the step that makes the whole-body per-cell model runnable. Every agent (a real Xenium cell) must carry copies/cell for every antibody target. Xenium panels are small, so off-panel targets are imputed onto every cell from a matched single-cell reference (Tabula Sapiens for organs, cancer-matched atlas for tumors). Full method: MODEL_DOCUMENTATION_FINAL/subsystems/T2b_receptor_overlay_imputation.md.

The three tiers:

  • Tier 1 native (CP10K-log1p) — gene on panel → normalize to 1e4 counts, log1p. Deterministic/byte-exact.
  • Tier 2 scVI overlay (topk_spread, K=15) — gene off panel → co-embed query+reference with scVI

(n_latent=20, 60 epochs CPU, seed 0), softmax-weighted draw over 15 nearest reference cells, transfer the reference cell's expression. ref_cell_assign records the chosen reference cell per query cell.

  • Tier 3 RNA→copies — HPA-IHC anchor (Path A) or HPA nCPM→Glassman (Path B) × per-cell rank modifier.

# 2a. Tier-1 native extraction — reproduce CP10K-log1p exactly (byte-exact check).
import scipy.sparse as sp, h5py
def cp10k_log1p(M_genes_by_cells):
    Mc=M_genes_by_cells.tocsc().astype(np.float32)
    lib=np.asarray(Mc.sum(0)).ravel(); lib[lib==0]=1
    Mc=Mc.multiply(sp.csr_matrix(1e4/lib)).tocsc(); Mc.data=np.log1p(Mc.data)
    return Mc
print("Tier-1 native extraction: CP10K-log1p (build_cancer_abm.py:50). "

# 2b. Tier-2 scVI overlay — the algorithm, exactly as in build_cancer_abm.py (steps 3-4).
# Shown as the canonical recipe. Full scVI re-train is ~minutes/organ on CPU; the deterministic
# assignment math below is the load-bearing kernel that transfers off-panel receptors.
from sklearn.neighbors import NearestNeighbors
def topk_spread_overlay(Zref, Zq, ref_expr_present, K=15, seed=0):
    """Zref/Zq: scVI latent (n_ref x 20, n_q x 20). ref_expr_present: (n_ref x G) normalized expr of
    the off-panel genes present in the reference. Returns (assign, transferred_expr)."""
    rng=np.random.RandomState(seed)
    nn=NearestNeighbors(n_neighbors=K).fit(Zref); dist,ind=nn.kneighbors(Zq)
    tau=np.median(dist[:,1:])+1e-9; w=np.exp(-dist/tau); w/=w.sum(1,keepdims=True)
    cum=np.cumsum(w,1); r=rng.rand(len(Zq),1); pick=(r<cum).argmax(1)
    assign=ind[np.arange(len(Zq)),pick]
    return assign, ref_expr_present[assign]
print("Tier-2 overlay kernel defined (topk_spread K=15, softmax bandwidth=median NN dist, seed=0).")
print("Canonical driver: tumor_builds/build_cancer_abm.py steps 3-4; census_version=2025-11-08.")
