FIG 05

Literature → a queryable pharmacology database

To validate the QSP model against real drugs, Claude read the clinical PK/PD literature and FDA labels and built a structured, cited database (mab_tce_pkpd.sqlite) — digitizing time–concentration curves from published figures and extracting binding kinetics, every value carrying its source. A pharmacometrician needs 1–3 weeks for this with no citation trail; here it ran in ~12 active hours, fully sourced.
